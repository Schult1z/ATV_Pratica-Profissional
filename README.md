# 🏟️ Sistema de Reserva de Quadras

Plataforma web desenvolvida para o gerenciamento e agendamento de quadras e espaços esportivos de forma prática e organizada. O sistema permite que clientes realizem reservas online, evitando conflitos de horários e melhorando o controle das agendas.

---

## 🚀 Funcionalidades

- Cadastro e login de usuários  
- Reserva de quadras online  
- Visualização de horários disponíveis  
- Área administrativa para controle de reservas  
- Gerenciamento de usuários  

---

## 👥 Tipos de Usuário

- **Cliente:** pode consultar horários e realizar reservas  
- **Administrador:** gerencia reservas, usuários e disponibilidade das quadras  

---

## 🛠️ Tecnologias Utilizadas

- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL
